# Toma y lee

Biblioteca colaborativa online.

## Qué es

Toma y lee es una plataforma web donde la comunidad puede compartir libros, reseñas y recursos de lectura vinculados a la fe, la formación y la interioridad.

## Funcionalidades

- Registro e inicio de sesión con confirmación por mail
- Catálogo público de libros
- Publicación de libros con link, PDF y portada
- Categorías
- Reseñas y valoración
- Reportes de libros y reseñas
- Panel de administración
- Auto-ocultado por reportes
- Perfil de usuario para gestionar libros propios

## Stack

- Next.js
- Supabase
- Vercel
- TypeScript
